# Mobilna Diagnostika – Futuristic Auto Diagnostics Platform

Този проект представлява Next.js приложение с App Router и двуезична (BG/EN) поддръжка за уебсайт на мобилна автодиагностика. Интерфейсът е изграден с Tailwind CSS и включва футуристичен дизайн с glassmorphism, mega меню, командна търсачка и тъмен/светъл режим.

## Основни функционалности
- Мега меню с водещи автомобилни сегменти (европейски, японски/корейски, американски внос).
- Командна търсачка с бърз достъп (Ctrl/⌘+K) до ключови услуги.
- Поддръжка на български и английски език, бърз суич за светла/тъмна тема.
- Специални секции за американски автомобили, хибриди/EV, карта на покритие и информационни ресурси.
- Интегрирана Viber CTA за чат и форма за контакт.

## Стартиране на проекта
```bash
npm install
npm run dev
```

Приложението ще бъде достъпно на http://localhost:3000. За production билд използвайте `npm run build` и `npm start`.

## Структура
- `src/app` – App Router страници и глобални стилове.
- `src/components` – UI компоненти, секции и помощни елементи (тема, търсене, чат).
- `src/data/translations.ts` – преводи и контент за двата езика.
- `src/lib` – помощни функции и контекст за преводите.
